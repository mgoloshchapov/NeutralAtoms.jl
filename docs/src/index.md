# NeutralAtoms.jl

[![Build Status](https://github.com/mgoloshchapov/NeutralAtoms.jl/actions/workflows/CI.yml/badge.svg?branch=main)](https://github.com/mgoloshchapov/NeutralAtoms.jl/actions/workflows/CI.yml?query=branch%3Amain)
[![docs](https://img.shields.io/badge/docs-latest-blue.svg)](https://mgoloshchapov.github.io/NeutralAtoms.jl/dev/)

`NeutralAtoms.jl` simulates neutral-atom experiments in optical tweezers. The
package is organized around the same experimental workflow that appears in the
single-atom Rydberg-excitation study of
[de Léséleuc et al. (2018)](https://arxiv.org/abs/1802.10424) and the
blockade-mediated global-pulse gate study of
[Levine et al. (2019)](https://arxiv.org/abs/1908.06101):

- characterize a Gaussian tweezer and sample thermal atomic motion
- estimate temperature with release-and-recapture curves
- model effective two-photon Rydberg excitation with Doppler shifts
- add stochastic laser phase noise and Lindblad decay channels
- simulate blockade-mediated two-qubit phase gates and analyze parity/fidelity

## Installation

Install the package from the GitHub repository:

```julia
using Pkg
Pkg.add(PackageSpec(url="https://github.com/mgoloshchapov/NeutralAtoms.jl"))
```

## What The Package Models

The package uses a five-level single-atom basis containing the two
computational ground states, the target Rydberg state, an intermediate optical
state, and a loss state. On top of that microscopic model it provides:

- Gaussian-beam optics helpers for tweezer and excitation beams
- thermal Monte Carlo sampling of coordinates and velocities
- release-and-recapture simulation for temperature inference
- time-dependent master-equation simulation of single-atom two-photon Rydberg
  excitation
- optional laser phase-noise trajectories sampled from a spectral-density model
- blockade-mediated two-atom controlled-phase simulations
- helper utilities for probabilities, parity scans, and error budgets

## Scientific Context

The single-atom tools are most useful when you want to reason about the three
main damping mechanisms emphasized in
[arXiv:1802.10424](https://arxiv.org/abs/1802.10424):

- Doppler dephasing from finite-temperature motion
- spontaneous emission through the intermediate state
- laser phase noise

The two-qubit tools are aimed at the blockade regime described in
[arXiv:1908.06101](https://arxiv.org/abs/1908.06101), where a controlled-phase
gate is generated by two global Rydberg pulses separated by a laser-phase step.
`CZLPConfig`, `simulation_czlp`, `get_parity_fidelity`, and `get_cz_infidelity`
are the key entry points for that workflow.
